## Prerequisites

- Node.js 20+ ([nodejs.org](https://nodejs.org))
- npm (bundled with Node.js) or pnpm
- Git

## Quickest Path

```bash
git clone https://github.com/BlackRoad-OS/blackroad-os-docs.git
cd blackroad-os-docs
npm install
npm run start
```

Dev server runs at **http://localhost:3000**.
